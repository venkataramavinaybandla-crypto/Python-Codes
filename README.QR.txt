# QR code Maker

A Python-based utility that generates QR codes from user-provided input such as text, URLs, or custom data. The project focuses on understanding external library usage, data encoding, and basic automation through Python.

---

## Objective

To build a simple and reliable QR code generation tool while learning how Python interacts with third-party libraries to produce real-world outputs.

---

## Features

- Generate QR codes from text or URLs  
- Save generated QR codes as image files  
- Simple and fast execution  
- Minimal user interaction  

---

## Technologies Used

- Python  
- qrcode library  
- Pillow (image processing)  

---

## How to Run

```bash
python qr_code_generator.py
